セクション 2
Claude Code とは何か
ChatGPT 等との違いを理解する
動詞で覚える3つの違い
ChatGPT「相談する」→ 会話でアドバイス・文章提案
Copilot「補完する」→ コード・文書の入力補完
Claude Code「実行する」→ ファイルを読む・編集・保存する
Claude Code は「相談」ではなく「実行」する
Claude Code とは8 / 24
ファイルを直接操作できる唯一のツール
  • ファイルを開いて読む・内容を処理する
  • 新しいファイルを作って保存する
  • 複数ファイルをまたいで一括処理する
  • 指示した作業を自動で代行する
Claude Code とは9 / 24
CLAUDE.md と料金について
  • CLAUDE.md：Claude Codeへの「事前指示書」となる設定ファイル
  • 一度書けば毎回同じ指示を繰り返さなくて済む
  • 詳しい使い方は第2回で実際に体験する
  • 料金：Claude Pro（$20/月〜）以上が必要

安全ルール12 / 24
ルール①　機密データを渡さない
  • 指示・ファイル内容はAnthropicサーバーに送信される
  • 顧客リスト・個人情報・社外秘資料はNG
  • 判断基準：「漏れたら困るか？」と自問する
  • 迷ったら使わない・上長に確認が鉄則
迷ったら使わない
安全ルール13 / 24
ルール②　確認画面を読まずに進まない
  • ターミナルからの削除はゴミ箱に入らない
  • 確認画面が最後のチャンス：必ず読む
  • 対策：作業前に重要ファイルのコピーを取る
  • 「迷ったら Reject（拒否）」が基本スタンス
安全ルール14 / 24
ルール③　作業専用フォルダを使う
  • デスクトップに「claude_work」フォルダを作成
  • ファイルはコピーしてから入れる（移動禁止）
  • claude_work の外のファイルには触れさせない
  • 「実験エリア」と「本番エリア」を分けるイメージ
